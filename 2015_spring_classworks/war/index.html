1 Сколько влезет в холодильник Пользователь выбирает холодильник и продукты, сервлет расчитывает какие продукты влезут в выбранный холодильник, а какие нет. В рамках данного задания студент знакомится с классами, учится использовать классы в сервлете, методы и поля классов. Google AppEngine, html, css, сервлеты, классы, jQuery. 2 Чтение из файла (открытые данные) Отображение информации какие проекты планировок запланированы для района по его коду ОКАТО. Данные берутся с Портала открытых данных Петербурга (база районов) и база территорий, по которым утверждены проекты планировок). В рамках данного задания студент знакомится с чтением из файла (файл находится на сервере), подключает библиотеку OpenCSV для работы с файлом CSV, создает модель данных (из двух объектов District и Plan), использует коллекции и цикл Foreach.
